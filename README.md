# 프로젝트 개요

**1. 프로젝트명 : Acontainer**

**2. 주제 : 창고 중개 플랫폼 개발**

**3. 배경**
- 공급과잉으로 인한 수요 불균형 및 창고 공실률 증가로 인한 문제를 해결하는 목적으로 창고 중개 플랫폼을 개발하였습니다.

**4. 목적**
-  ﻿본 프로젝트는 물류 시스템의 효율성을 향상시키기 위해 창고의 위치 정보를 관리하고 창고 주인이 자신의 창고를 해당 지역에 등록함으로써 창고 공실률을 감소시키는 것을 목적으로 합니다
